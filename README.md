## 주요기능
----------
* 오늘 날씨 보기
* 현재 위치 확인해서 날씨 보여주기
* 검색 도시 날씨 확인하기
* 검색 안될시 에러 메시지 출력
* 검색된 도시 7일간의 날씨 출력
